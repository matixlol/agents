# Ahrefs

Use the Ahrefs MCP server for requests about domains, backlinks, keywords, rankings, organic visibility, brand visibility, or comparable SEO research.

## Rules

- Prefer Ahrefs MCP tools over generic web browsing when the task is specifically about Ahrefs SEO data.
- Be explicit when a conclusion is an inference from Ahrefs metrics rather than a direct Ahrefs field.
- Keep requests focused because Ahrefs usage consumes the account's API allowance.
- If Ahrefs tools are unavailable or the user is not authenticated, say that directly and continue with the best fallback.

## Typical tasks

- Compare domains or URLs
- Find top keywords or ranking changes
- Inspect backlink profiles and referring domains
- Check brand or search visibility trends
